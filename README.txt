To install this plugin, simply upload the hit_counter.php file in the 
included folders to the same path in your MyBB installation.

That means...

1) In this zip find hit_counter.php in inc/plugins/hit_counter.php

2) Place this file in the inc/plugins/ directory of your MyBB installation.
